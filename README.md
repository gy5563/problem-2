# Problem 2: Fancy Form

### Description

This project is a cypto currency converter built with javascript. The price data are based on `https://interview.switcheo.com/prices.json`

### Installation

Make sure you have Node.js (v16+) and npm installed.

Clone the repository and install dependencies:

```console
npm install
```
### Development

Start the development server:

```console
npm run dev
```
